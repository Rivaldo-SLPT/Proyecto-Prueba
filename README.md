# Sistema de Gestión de Pedidos - Cusco 🍕

## 1. Descripción del Proyecto
Este proyecto es el resultado del **Laboratorio 1 de Ingeniería de Software**[cite: 1]. [cite_start]Consiste en la configuración inicial de un sistema web para una startup en la ciudad de Cusco dedicada a la gestión de pedidos de comida local[cite: 16].

## 2. Objetivos del Laboratorio
* Configurar un entorno de desarrollo moderno[cite: 3, 6].
* Aplicar el control de versiones utilizando Git en un proyecto básico[cite: 3].
* Gestionar repositorios locales y remotos a través de GitHub[cite: 7].

## 3. Herramientas Utilizadas
* **Editor de Código:** Visual Studio Code o IntelliJ[cite: 12].
* **Control de Versiones:** Git[cite: 13].
* **Repositorio Remoto:** GitHub[cite: 14].
